# 🚛 途吉天下卡车管理系统

> 面向物流公司、个体卡车司机的 GPS 定位 + 轨迹回放 + 油耗统计 + 电子围栏系统

## 产品定位

让司机和物流公司能随时看到车在哪、怎么跑的，油耗多少。

## 核心功能

### 📱 客户端（司机/车主）

| 功能 | 说明 |
|------|------|
| 📍 **实时定位** | 打开手机就能看到车在哪 |
| 🗺️ **轨迹回放** | 查看历史行驶路线 |
| ⛽ **油耗统计** | 记录加油，自动算平均油耗 |
| 🔲 **电子围栏** | 设置区域，进出自动提醒 |
| 🚚 **车辆管理** | 添加/管理车辆信息 |
| 👤 **账号管理** | 登录、注册、个人信息 |

### 🖥️ 平台管理后台（管理员）

| 功能 | 说明 |
|------|------|
| 👥 **用户管理** | 管理员/司机/车主账号管理 |
| 🚚 **车辆管理** | 所有车辆信息管理 |
| 📊 **数据统计** | 在线车辆、活跃用户、轨迹数据 |
| 📢 **消息推送** | 围栏告警、系统通知推送 |
| ⚙️ **系统设置** | 平台配置、权限管理 |
| 📋 **操作日志** | 管理员操作记录 |

## 用户群体

- **物流公司** - 管多辆车，查看所有车的位置
- **个体司机** - 记录油耗、查看自己车的轨迹
- **平台管理员** - 管理系统、查看数据

## 技术栈

- **小程序端**: UniApp (Vue3 + TypeScript) → 微信/支付宝小程序
- **管理后台**: Vue3 + Element Plus
- **后端**: NestJS (Node.js + TypeScript)
- **数据库**: MySQL

## 开发指南

### 小程序端开发

```bash
cd frontend
npm install
npm run dev:mp-weixin
```

### 管理后台开发

```bash
cd admin
npm install
npm run dev
```

### 后端开发

```bash
cd backend
npm install
npm run start:dev
```
